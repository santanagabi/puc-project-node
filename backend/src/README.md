# API de Usuários 🚀

API de usuários com Node.js, Express, Prisma e MongoDB.

## 🛠 Tecnologias

- Node.js
- Express
- Prisma
- MongoDB

## 📚 Endpoints

- **Criar usuário**: `POST /usuarios`
- **Listar usuários**: `GET /usuarios`
- **Editar usuário**: `PUT /usuarios/:id`
- **Deletar usuário**: `DELETE /usuarios/:id`
